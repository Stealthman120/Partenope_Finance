## Deploy su GitHub Pages (3 minuti)
1. Carica su una repository GitHub
2. Vai su Settings → Pages
3. Seleziona il branch `main` e la cartella `/ (root)`, poi salva.

## Installa su Android
Chrome → menu ⋮ → "Aggiungi alla schermata Home"

## Installa su iPhone
Safari → Condividi ↑ → "Aggiungi alla schermata Home"
